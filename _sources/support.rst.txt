=======
Support
=======

.. contents::
   :depth: 1
   :local:

Please use one of the following for general support questions:

* The `python-win32`_ mailing list. Note that you must subscribe to the list before posting. 
* `Stack Overflow`_ with the :kbd:`[pywin32]` tag
* A search engine :samp:`site:stackoverflow.com pywin32 {search term}`
   - `DuckDuckGo`_
   - `Microsoft Bing`_
   - `Google`_

.. _python-win32: http://mail.python.org/mailman/listinfo/python-win32
.. _Stack Overflow: https://stackoverflow.com/questions/tagged/pywin32
.. _DuckDuckGo: https://duckduckgo.com/?q=site%3Astackoverflow.com+pywin32+help+me
.. _Microsoft Bing: https://www.bing.com/search?q=site%3Astackoverflow.com+pywin32+help+me
.. _Google: https://www.google.com/search?q=site%3Astackoverflow.com+pywin32+help+me

Please, do not use the GitHub issue tracker for general support.

Feel free to open an `issue`_ for all bugs (or suspected bugs). `Pull requests`_ for all bugs or features are also welcome.

.. _issue: https://github.com/mhammond/pywin32/issues
.. _Pull requests: https://github.com/mhammond/pywin32/pulls
